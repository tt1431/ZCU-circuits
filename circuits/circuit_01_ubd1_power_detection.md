# 电路 01 — UBD1 双三极管电子开关

> **项目**：ZCU 区域控制单元 | **子系统**：电源管理
> **核心架构**：PNP + NPN 双三极管级联 → 受控电子开关
> **设计日期**：2026

---

## 📌 一句话说清

**用双三极管搭了一个"软件可控的电子开关"**——MCU 发信号，NPN 驱动 PNP，把 UBD1 电源电压引出来送到 ADC 采样。不检测的时候开关断开，零功耗。

---

## ⚡ 核心原理：双三极管电子开关

这个电路的本质不是"电源检测"，而是**一个受控电子开关 + 分压网络**。

### 开关结构

```
UBD1 ──→ [PNP 发射极] ←─┐
                        │  R3000 (10K) ← [NPN 集电极] ←┐
         [PNP 集电极] ──→ Vout（到分压/ADC）            │
                       DO_UBCTRL ──→ [NPN 基极]
                                     [NPN 发射极] → GND
```

**两个三极管的分工**：

| 三极管 | 角色 | 类比 |
|:--|:--|:--|
| **NPN** | **驱动级（控制端）** | 继电器线圈——接收 MCU 的小信号 |
| **PNP** | **功率级（开关主体）** | 继电器触点——接通/断开 UBD1 电源通路 |

### 工作逻辑

**开关闭合（开启检测）**：DO_UBCTRL = H

```
DO_UBCTRL = H (3.3V)
  → NPN 基极高电平 → NPN 导通（V_BE ≈ 0.7V）
  → NPN 集电极 ≈ 0V（V_CE(sat) ≈ 0.1V）
  → 通过 R3000，PNP 基极被拉到 ≈ 0V
  → PNP：V_EB = UBD1 - 0 ≈ UBD1（例如 12V），远大于 0.7V
  → PNP 导通！发射极 = UBD1，集电极输出 ≈ UBD1
  → 分压网络工作 → ADC 读到电压
```

**开关断开（空闲）**：DO_UBCTRL = L

```
DO_UBCTRL = L (0V)
  → NPN 基极低电平 → NPN 截止
  → NPN 集电极高阻 → PNP 基极通过结漏电流浮高至 ≈ UBD1
  → PNP：V_EB ≈ 0 → PNP 截止
  → 无输出 → ADC 读 0V → 零功耗
```

> 💡 **设计精髓**：开关只有"测"的时候才闭合，平时断开——车载低功耗场景的核心要求。

### 分压比计算

$$
V_{ADC} = V_{PNP\_out} \times \frac{R_{3010}}{R_{3005} + R_{3010}}
       = V_{PNP\_out} \times \frac{4.7}{24.3 + 4.7}
       \approx V_{PNP\_out} \times 0.162
$$

PNP 导通后 $V_{PNP\_out} \approx V_{UBD1}$（忽略饱和压降），因此：

$$
V_{UBD1} \approx \frac{V_{ADC}}{0.162}
$$

**典型值**：

| UBD1 | ADC 电压 | 12-bit ADC code (3.3Vref) |
|:--|:--|:--|
| 5V | 0.81V | ~1005 |
| 12V | 1.94V | ~2410 |
| 18V | 2.92V | ~3623 |
| 24V | 3.89V | **超量程 ❌** |

> ⚠️ 分压比 0.162 是为最高 12V 设计的。若 UBD1 = 24V，需调 R3005 → 68K 或 R3010 → 2.2K。

### BOM

| 编号 | 类型 | 参数 | 封装 | 说明 |
|:--|:--|:--|:--|:--|
| Q3001 | 双三极管 | NSVMUN5333DW1T1G | SOT-363 | NPN+PNP 合封，带偏置电阻 |
| R3000 | 电阻 | 10KΩ | 0402 | NPN 集电极 → PNP 基极 |
| R3806 | 电阻 | 4.7KΩ | 0402 | NPN 基极下拉 |
| R3005 | 电阻 | 24.3KΩ 1% | 0402 | ADC 分压上臂 |
| R3010 | 电阻 | 4.7KΩ 1% | 0402 | ADC 分压下臂 |
| C3001 | 电容 | 10nF 50V | 0402 | ADC 滤波 |
| TP3005/3007/3016 | 测试点 | — | — | 生产调试 |

### Q3001 选型

**型号**：ON Semi **NSVMUN5333DW1T1G**

**要点**：
- **NSV 前缀** = AEC-Q101 车规认证（-40 ~ +125°C）
- **MUN5333** = 内置偏置电阻（BRT），简化外围
- **DW1** = SOT-363 双管合封（6 脚），一颗料顶两颗，省 PCB 面积
- **一颗料搞定整个电子开关！**

### R3000 — PNP 基极驱动

$$
I_{B(PNP)} \approx \frac{V_{UBD1} - V_{CE(sat,NPN)} - V_{BE(PNP)}}{R_{3000}}
            \approx \frac{12 - 0.1 - 0.7}{10\text{K}} = 1.12\text{ mA}
$$

PNP 集电极负载 ~29KΩ（分压网络），1.12mA 基极驱动绰绰有余。

### R3806 — 基极下拉

4.7K 确保 MCU 复位/高阻态时 NPN 基极被可靠拉低 → PNP 截止 → 不会误导通。

$$
I_{pulldown} = \frac{3.3}{4.7\text{K}} \approx 0.7\text{ mA}
$$

这个电流对 MCU GPIO 完全可承受。

### C3001 — RC 低通滤波

等效输出阻抗：

$$
R_{eq} = R_{3005} \parallel R_{3010}
       = \frac{24.3 \times 4.7}{24.3 + 4.7}
       \approx 3.94\text{K}
$$

截止频率：

$$
f_{-3dB} = \frac{1}{2\pi \times R_{eq} \times C_{3001}}
         \approx \frac{1}{2\pi \times 3940 \times 10^{-8}}
         \approx 4.0\text{ kHz}
$$

4kHz 足以滤除开关噪声，响应时间 ~0.2ms（5τ），不影响检测速度。

### 精度预算

| 误差源 | 值 | 贡献 |
|:--|:--|:--|
| R3005 公差 | ±1% | ±1% |
| R3010 公差 | ±1% | ±1% |
| PNP V_CE(sat) | ~0.1V | 12V 时 <1% |
| ADC 量化 | 12-bit | 可忽略 |
| **总 RSS** | — | **±1.4%** |
| 12V 时误差 | — | **±0.17V** |

## 🛠 调试指南

### 故障 1：ADC 恒为 0

| 检查点 | 正常值 | 异常原因 |
|:--|:--|:--|
| TP3005（UBD1） | 被测电压 | 电源没来 |
| NPN 基极 | DO_UBCTRL = H 时 ~0.7V | R3806 短路 / GPIO 没输出 |
| NPN 集电极 | NPN 导通时 ≈ 0.1V | NPN 坏 / R3000 开路 |
| PNP 基极 | PNP 导通时 ≈ 0.1V | R3000 虚焊 |
| TP3016 | 分压值 | 分压网络断 |

> 🔑 **口诀**：读数归零先看 UB，NPN 不通查下拉，PNP 不导通查 R3000。

### 故障 2：ADC 读数跳动

- C3001 虚焊/缺失 → 无滤波，换 100nF 试试
- 走线平行大电流开关信号 → 耦合噪声 → 软件加滑动平均
- ADC 参考电压不稳 → 检查 MCU 电源

### 故障 3：DO_UBCTRL = L 时 ADC 仍有读数

- NPN CE 漏电？
- R3806 开路？NPN 基极浮空，噪声触发误导通
- PNP CE 漏电？（高温下可能性增大）

### 调试口诀

> 读数归零先看使能，跳动厉害先查电容。
> NPN 不通看下拉，PNP 不导通查基极。
> 空闲还有电压？下拉电阻焊上了没！

>

### 可能会被追问

**Q1：为什么用 PNP+NPN 级联而不是单管？**

A：如果单用一个 NPN 直接串在 UBD1 通路上——NPN 做高边开关需要基极电压高于发射极，MCU 的 3.3V GPIO 无法直接驱动 12V 电源轨上的 NPN。

PNP 天然适合做高边开关（发射极接电源，基极拉低就导通），但 MCU 的 3.3V GPIO 拉不低 12V——中间加一级 NPN 做电平转换，NPN 导通后集电极 ≈ 0V，轻松拉低 PNP 基极。

光耦方案贵一个数量级，且 UBD1 和 MCU 共地不需要隔离。

**Q2：PNP 基极不需要上拉电阻吗？为什么断开时能可靠截止？**

A：理论上加一个从 PNP 基极到 UBD1 的上拉电阻（如 100K）更可靠。实际中 NPN 截止后 PNP 基极通过结漏电流浮高至接近 UBD1，V_EB < 0.7V，PNP 可靠截止。实验室验证可以，量产建议加上拉。

**Q3：分压比为什么是 0.162？**

A：UBD1 最大 12V，分压后不超过 ADC 参考电压 3.3V。$12 \times 0.162 = 1.944\text{V} < 3.3\text{V}$。同时尽量用大分压比以获得更好分辨率——比 0.25 牺牲了量程但换来了精度。

**Q4：如果 UBD1 是 24V 怎么办？**

A：改分压网络：R3005 → 68K（分压比 4.7/(68+4.7) ≈ 0.065），24V 分到 ~1.55V。R3000 也可能需要从 10K 调大（限制 PNP 基极电流）。或者直接用 4 电阻精密分压 + 轨到轨运放跟随，彻底解决量程问题。
