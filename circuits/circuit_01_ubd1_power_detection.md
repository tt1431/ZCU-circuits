# 电路 01 — UBD1 双三极管电子开关

> 用两颗三极管搭了一个受控电子开关，把 12V 电源电压引出来送给 MCU 的 ADC 采样。不测的时候断开，零功耗。

## 电路怎么工作的

跟着信号走一遍就清楚了。

### 开关闭合（MCU 发起检测）

```
DO_UBCTRL → H (3.3V)
  │
  ├─→ Q1(NPN) 基极 = 3.3V  →  Q1 导通  →  Q1 集电极拉到 ≈0V
  │
  ├─→ R2(10K) 把 Q1 集电极的 0V 传到 Q2(PNP) 基极
  │
  └─→ Q2 基极 ≈ 0V，发射极 = UBD1(12V)
         → V_EB = 12V  >>  0.7V  →  Q2 导通
         → Q2 集电极输出 ≈ 12V
         → R3/R4 分压（×0.162）→ C1 滤波 → MCU ADC
```

### 开关断开（空闲）

```
DO_UBCTRL → L (0V)
  │
  └─→ Q1 基极 = 0V  →  Q1 截止
         → R2 上端高阻  →  Q2 基极通过漏电流浮高至 ≈ UBD1
         → V_EB ≈ 0  →  Q2 截止
         → 无输出，不耗电
```

| 控制信号 | Q1 (NPN) | Q2 (PNP) | ADC |
|:--|:--|:--|:--|
| DO_UBCTRL = H | 导通，集电极 ≈0V | 导通，输出 ≈UBD1 | 读到分压值 |
| DO_UBCTRL = L | 截止 | 截止 | 0V |

## 为什么必须用两个三极管？

MCU 的 GPIO 只有 3.3V，要控制的是 12V 电源轨——**电平不匹配**。

| 如果只用一颗 | 为什么不行 |
|:--|:--|
| 单 NPN 串在 12V 通路上 | 发射极在 12V，基极需 ~12.7V 才导通，MCU 做不到 |
| 单 PNP 串在 12V 通路上 | 截止需基极 ≈ 12V，MCU 最高只能出 3.3V，关不断 |

**加一级 NPN 就解决了**：NPN 的基极只要 0.7V 就导通——3.3V 绰绰有余。导通后 NPN 集电极 ≈ 0V，这个 0V 足够把 PNP 基极拉低，PNP 在 12V 轨上可靠导通。

> 第一级（NPN）做电平转换，第二级（PNP）做功率开关。各司其职。

## 每个器件是干什么的

### Q1 + Q2 = NSVMUN5333DW1T1G（一颗料）

这是一颗**双三极管**，SOT-363 封装里塞了一个 NPN 和一个 PNP。

- AEC-Q101 车规，-40~125°C
- 内置偏置电阻（BRT），外围少两个电阻
- V_CEO = 50V，I_C = 100mA——对本电路 0.4mA 的负载完全是杀鸡用牛刀

> 为什么不用 MOSFET？负载只有 ~29KΩ 分压网络，电流不到 0.5mA。BJT 几分钱搞定，MOSFET 贵一个数量级。场景够用了就别上贵的。

### R2 = 10K（级联电阻）

NPN 集电极到 PNP 基极。NPN 导通时把 PNP 基极拉低。

$$
I_{B(PNP)} \approx \frac{12 - 0.1 - 0.7}{10K} = 1.12\text{mA}
$$

PNP 的集电极负载是 29K 分压网络（~0.4mA），基极电流 1.12mA 驱动它绰绰有余。

### R1 = 4.7K（下拉）——为什么是 4.7K？

MCU 复位期间 GPIO 是高阻态。如果没有下拉，NPN 基极浮空——噪声、漏电流都可能让它误导通 → PNP 误导通 → 误检测。

4.7K 是一个三角形折中：

- **太小（1K）**：GPIO 高电平时 $I = 3.3V/1K = 3.3mA$, 浪费驱动能力
- **太大（100K）**：下拉太弱，压不住噪声
- **4.7K**: $I = 0.7mA$, 够强压制噪声，GPIO 轻松承受

### R3 = 24.3K / R4 = 4.7K（分压网络）

把 12V 缩到 ADC 能接受的范围: $$
V_{ADC} = V_{UBD1} \times \frac{4.7}{24.3 + 4.7} \approx V_{UBD1} \times 0.162
$$

| UBD1 | ADC 电压 | 12-bit code (3.3Vref) |
|:--|:--|:--|
| 5V | 0.81V | ~1005 |
| 12V | 1.94V | ~2410 |
| 18V | 2.92V | ~3623 |

> 分压比 0.162 是为 12V 系统设计的。如果 UBD1 是 24V，把 R3 换成 68K 就行了。

### C1 = 10nF（滤波）

和分压网络的等效阻抗（~3.94K）构成 RC 低通: $$
f_{-3dB} = \frac{1}{2\pi \times 3.94K \times 10nF} \approx 4.0\text{kHz}
$$

4kHz 滤掉开关噪声足够了，响应时间 ~0.2ms（5τ），不影响检测速度。

### 精度

| 误差源 | 贡献 |
|:--|:--|
| R3 ±1% | ±1% |
| R4 ±1% | ±1% |
| ADC 12-bit | 可忽略 |
| **总 RSS** | **±1.4%**（12V 时 ±0.17V） |

## 三极管基础知识速查

### 饱和压降

硅管的 $V_{BE}$ 永远是 ~0.7V 左右导通，深度饱和到 ~0.85V 。
$V_{CE(sat)}$ 跟负载电流走——mA 级可以做到 0.05~0.1V 。

| 参数 | 小信号管 | 功率管 | 达林顿管 |
|:--|:--|:--|:--|
| $V_{BE(sat)}$ | 0.7~0.85V | 0.8~1.2V | 1.2~1.4V |
| $V_{CE(sat)}$ | 0.1~0.3V | 0.3~1V | 0.7~1.5V |
| 电流 | mA 级 | A 级 | A 级 |
| $h_{FE}$ | 100~300 | 20~50 | 1000~30000 |
| 典型型号 | 2N3904, MUN5333 | TIP31C, 2N3055 | TIP122, ULN2003 |

本电路的 MUN5333 属于**小信号管**——负载 0.4mA，用功率管是杀鸡用牛刀，用达林顿管反而压降大。

## 调试

### ADC 恒为 0

从后往前查：UBD1 有电？→ Q1 基极有控制信号？→ Q1 集电极 = 0V？→ Q2 基极被拉低？→ 分压输出正常？

### ADC 读数跳动

先查 C1 焊上了没。没有 C1 的滤波，几百 mV 级别的跳动很正常。软件再加个滑动平均。

### 空闲时 ADC 还有读数

一般是 Q1 基极浮空 → 噪声触发误导通。查 R1 下拉电阻是不是开路了。

### 选型 checklist

以后选三极管，脑子里过这几个：

| 参数 | 问自己 |
|:--|:--|
| $V_{CEO}$ | 最高工作电压？留 2 倍余量 |
| $I_C$ | 负载电流？留 3~5 倍 |
| $V_{CE(sat)}$ | 压降对精度有影响吗？越小越好 |
| 封装 | 要双管合封吗？要内置电阻吗？ |
| 车规 | 车载项目必须有 AEC-Q101 |
